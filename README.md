# 🛡️ ScamAdvisor - AI-Based Website Trust Analyzer

## 🚀 Overview

ScamAdvisor is a tool that analyzes websites and detects whether they are safe or malicious. It helps users identify phishing websites and avoid online scams.

---

## 📚 Features

- 🔍 Website Analysis  
- 🚨 Phishing Detection  
- 🔐 SSL Verification  
- 📊 Risk Score Generation  
- 🖥️ User-Friendly Interface  

---

## ⚙️ How It Works

1. User enters a website URL  
2. The system extracts features like URL length, HTTPS usage, and patterns  
3. A Machine Learning model analyzes the data  
4. The system predicts whether the site is safe or malicious  
5. The result is displayed with a risk score
